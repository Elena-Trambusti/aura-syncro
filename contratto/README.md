## Generare PDF in 2 minuti

1. Doppio click su `CONTRATTO_RIEPILOGO_ESECUTIVO.html`
2. Chrome → `Ctrl+P` → **Salva come PDF**
3. Per il contratto lungo: apri il `.md` in Word/Google Docs → Esporta PDF

## Firma consigliata

- **YouSign** o **DocuSign** (B2B, audit trail)
- **Aruba Sign / InfoCert** (firma elettronica qualificata)
